# Learner profile & mission established

Chris is **rusty-but-broad**: has real prior exposure to electronics/circuits, programming/DSP,
marine/boating/diving, and physics/acoustics, but self-assesses as **medium-to-novice** and asked to be
treated that way. Implication: re-establish fundamentals at each step (don't assume retention), use correct
terminology freely, but never lean on assumed recall.

**Mission (confirmed):** build a hydrophone (~US$200–600 first build) and use it to **fully localize / track**
the resident sperm whales off Mauritius — not just detect them. Math depth: **balanced** (formulas + worked
examples + interactive calculators). Wants **interactive graphics/tools** in lessons. Delivery via **GitHub
Pages** at github.com/chrisfraser/hydrophone. See [[MISSION.md]].

This sets the whole curriculum arc: single hydrophone → record clicks → spectrograms → 2-element bearing →
multi-element TDOA localization. The summit (full localization) means later lessons must reach real array
geometry + hyperbolic positioning, not stop at detection.
